### <u>Venturebeat article analysis</u>
The following project aims to display how the use of scrapy, matplotlib and NLTK library 
can be put together to get some meaningful analysis. Here we crawl around 600+ articles 
of december 2018 from a tech-news site Venturebeat.com , put them in a large text file, 
and then read that file using NLTK library to build a hashmap of unique words, filtering 
out the non usefult stopwords.  

Links : [Actions Store](https://github.com/root-ansh/VenturebeatArticleAnalyser)
